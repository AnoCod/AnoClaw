# AnoClaw

AI desktop platform — users customize their own AI software by creating plugins, anywhere.

## Quick Start

Electron desktop app. Download and run the installer, or double-click `release9/win-unpacked/AnoClaw.exe`.

Build from source:

```bash
npm install
npm run build:all
npx electron-builder --win
```
Double-click `release9/win-unpacked/AnoClaw.exe`.

First launch opens a setup wizard to configure your LLM provider (API key, model, API URL).

## How it works

AnoClaw is an LLM-native desktop platform. AI agents use tools (Read, Write, Bash, WebSearch, etc.) to complete tasks. Users can install plugins to add new tools, pages, and capabilities — either written by others or generated by AI on the spot.

**The kernel is frozen at ~17 files** — process isolation, tool registry, event bus, LLM interface. Everything else is plugin territory. See `内核定义.md` for details.

## Features

- **Plugin system** — VSCode-style extensions in Worker Threads. Install, reload, or remove without restarting
- **Agent hierarchy** — MainAgent → Manager → Member → SubAgent organization
- **Streaming chat** — Real-time token streaming, tool cards, think panels
- **Workspace files** — File tree, preview, create/rename/delete operations
- **Background tasks** — Fire-and-forget with notification when complete
- **Memory system** — Persistent per-agent and team memory
- **Evolution tracking** — Pattern detection, quality scoring, session tagging

## Configuration

| File | Purpose |
|------|---------|
| `config/settings.yaml` | Server port, LLM provider, API key |
| `data/agents/` | Per-agent configs (model, tools, prompt) |
| `data/sessions/` | JSONL append-only conversation history |

First run: setup wizard auto-creates `config/settings.yaml` and `data/agents/ceo.json`.

## Tech Stack

TypeScript + Node.js `http` + `ws`. Frontend: vanilla DOM, no framework. Electron 42 desktop shell. DeepSeek API (OpenAI-compatible format).

## Build

```bash
npm run build:all      # Full build (server + frontend + CSS + plugins)
npm test               # Vitest (~500 tests)
```
